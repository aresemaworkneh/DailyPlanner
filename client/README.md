# Daily Planner App

This is a full-stack MERN app that helps you:

- Log daily activities  
- Plan tasks for the next day  
- Create a color-coded to-do list (with pastel themes)

## Tech Stack

- MongoDB  
- Express.js  
- React.js (with Tailwind CSS)  
- Node.js

## Getting Started

### Backend

```bash
cd server
npm install
npm run dev
